# Open / Private Boundary

- open license: Apache-2.0
- public workspace: `packages/**`, `docs/**`, `scripts/**`, `tests/**`, `reuse_log/**`
- private boundary: `private/**`
- browser profiles: never commit
- legacy: read-only asset reference

## Public artifacts

The open core may contain interface contracts, schema implementations, redline tests, local server code, Console shell code, and Obsidian bridge contracts.

## Private artifacts

Private artifacts include account aliases, tokens, browser profiles, cookies, paid database scripts, provider selectors that expose personal account patterns, private workflow cards, private prompts, and host-local vault state.

## Stage 0 rule

Stage 0 creates placeholders only. It does not include any real private data.
